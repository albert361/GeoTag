GeoTag Help: Saving changes

Image file meta-data is not updated until you Save your changes. If you have checked the Make backup files preference item you will find a copy of the unchanged file in your Trash. Backup copies are only made of files with geo-location changes. You can move the backup files elsewhere if you wish to keep them. Empty your Trash to throw them away.

The files are still listed by the application, allowing more changes to be made. The list of files may be cleared by selecting Clear image list menu item from the Edit menu.
