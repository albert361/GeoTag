GeoTag Help: Window layout

The main GeoTag window is sub-divided into three views as seen below. The view on the left list the files opened for processing. The view at the upper right displays the currently selected image. It is blank when no image is selected. The view at the bottom right is a google map. A marker is displayed on the map if an image is selected and that image contains geo-location information. If multiple images have been selected the image displayed and the location shown on the map will be that of the last image added to the selection.

The google map view can be panned and zoomed using the controls on the left side of the map. The map can be panned using the mouse. The marker can be moved by clicking on the map at the desired location or by dragging the marker to a different location.

There is one other window that should be noted. The bottom of the about window contains a code between hash marks, #v0.1-7-gdf5a3d5# in the following example. When reporting problems or requesting features please include that code in your correspondence. It identifies the exact build of GeoTag in use.
